## Starting server

1. Start the rabbitMQ server by running **rabbitmq-server.bat** from "...\RabbitMQ\rabbitmq_server-3.8.0\sbin"
2. Run scripts send, send2 and recieve.

```CMD
(from the dir where the .js files are located)
node send.js
node recieve.js
```
